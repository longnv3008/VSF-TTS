# Docs Index

Thu muc nay duoc to chuc de nguoi moi va AI agent co the vao du an nhanh, doc it file nhung van du context de tiep tuc lam viec.

## Read First

Neu la AI agent hoac nguoi moi vao repo, thu tu doc uu tien:

1. `docs/ai-context/project-memory.md`
2. `docs/ai-context/repo-map.md`
3. `docs/ai-context/work-rules.md`
4. `docs/phases/`

## Muc dich cua tung nhom tai lieu

- `ai-context/`: Bo nho nen, tom tat repo, quy uoc cap nhat, file doc nhanh cho AI.
- `phases/`: Chia tai lieu theo phase nghiep vu va roadmap ky thuat.
- `decisions/`: Ghi lai quyet dinh kien truc hoac trade-off quan trong.
- `runs/`: Nhat ky chay thu, handoff, bug trace, deployment note.
- `changelog/`: Tom tat thay doi cap cao de AI khong phai doc lai toan bo git diff.

## Quy tac cap nhat docs

Moi khi co thay doi lon, uu tien cap nhat toi thieu cac file sau:

1. `docs/ai-context/project-memory.md`
2. Phase lien quan trong `docs/phases/`
3. `docs/changelog/README.md`

Neu thay doi lien quan den kien truc, bo sung them mot file vao `docs/decisions/`.
